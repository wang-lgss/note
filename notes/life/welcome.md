# 欢迎使用笔记应用

这是一个功能完整的笔记应用，支持以下功能：

## 主要功能

### 📁 文件夹分类
- 在左侧侧边栏可以看到文件夹结构
- 点击文件夹可以查看该文件夹下的所有笔记
- 支持多级文件夹嵌套

### 🔍 搜索功能
- **全文搜索**：搜索笔记标题和内容
- **标签搜索**：按标签快速查找相关笔记

### 🏷️ 标签系统
- 为每个笔记添加多个标签
- 标签可以跨文件夹使用
- 支持按标签搜索和筛选

### ✍️ Markdown 编辑
- 支持完整的 Markdown 语法
- 实时预览功能
- 自动保存笔记

### 🤖 AI 补全
- 使用 Claude AI 自动补全笔记内容
- 只需输入关键字，AI 会帮你扩展内容
- 需要配置 Claude API 密钥

## 快速开始

1. 在左侧选择一个文件夹
2. 在笔记列表中选择或创建新笔记
3. 在编辑器中输入内容
4. 添加标签和元数据
5. 点击"保存"按钮保存笔记

## 使用 AI 补全

1. 在编辑器中输入关键字或不完整的内容
2. 点击"AI 补全"按钮
3. Claude AI 会自动生成补全内容
4. 补全内容会追加到原内容之后

**提示**：确保已在顶部配置了 Claude API 密钥。

## 笔记存储

所有笔记都存储在本地文件系统中，以 Markdown 格式保存。你可以随时备份 `notes/` 文件夹来保护你的数据。

祝你使用愉快！
